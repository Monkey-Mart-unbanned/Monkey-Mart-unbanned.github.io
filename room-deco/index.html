About RoomDeco

RoomDeco is the browser version of the cozy room design game, hosted on github.io so it loads without extra software. The full game runs in this page. There is no sign-up, no installer, and no pop-up before play. The design loop is simple: pick a room, place furniture, change colors, and save the layout you like best.

The browser build works on any device with a modern browser. Laptops, Chromebooks, desktops, tablets, and phones all run the same build. Touch controls handle drag and drop on mobile, and the cursor handles it on desktop. The game keeps your last design in memory, so closing the tab and coming back picks up where you left off.

You get the full game here, no price tag and no trial countdown. A free browser session is enough to design a small studio apartment or a full living room. The free version includes every room type and every furniture set that ships with the game.

RoomDeco is a casual design sim with no time pressure. Most actions are simple: drag a piece of furniture onto the floor plan, swap wall colors, and try a different layout until the room looks right. There is no fail state and no money counter, so the only goal is to make a space you want to look at.

RoomDeco is one of the cleanest design games on the web. You start with an empty room. You add a focal piece, layer in smaller items, and pick a small color palette so the room reads as one space. The game lets you save the layout and come back to it later, which makes it a good fit if you like to iterate.

How to play RoomDeco

Playing RoomDeco is straightforward. Open the embed above, pick a room from the menu, and start placing items. Drag furniture from the side panel onto the floor plan. Click a wall or floor to change its color. Use the undo button if a layout does not work. Save the design when you are happy with it.

Tips and tricks

Good RoomDeco designs usually start with a focal piece, like a sofa, bed, or dining table. Layer smaller items around it. Stick to a small color palette: two or three main colors plus one accent. Leave space between items so the room does not feel cramped. Match wood tones so different furniture pieces look like they belong together.
